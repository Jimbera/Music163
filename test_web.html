网易云音乐下载器测试

请在浏览器中打开 index.html 文件来使用完整功能。

使用说明：

  1. 确保 Flask 服务器正在运行（已在终端中启动）
  2. 在浏览器中打开 index.html 文件
  3. 在输入框中粘贴网易云音乐链接：
    • 歌单链接示例：https://music.163.com/m/playlist?id=2492467414&creatorId=621118881
    • 单曲链接示例：https://music.163.com/song?id=123456
  4. 选择音质
  5. 点击"开始处理"
  6. 等待下载完成后点击"保存"按钮

功能特点：

  • ✅ 支持歌单批量下载
  • ✅ 支持单曲下载
  • ✅ 支持混合输入（多行）
  • ✅ 深色/浅色主题切换
  • ✅ 下载历史记录
  • ✅ 多种音质选择
